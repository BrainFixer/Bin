# BrainFilxer Bin

Личный файловый архив для оперативного доступа к публичным ресурсам через GitHub Pages.

## Назначение
Репозиторий используется как легковесный веб-сервер для размещения файлов, которые необходимо быстро получить на внешних устройствах, серверах или при отладке систем.

## Доступ
URL проекта: [https://brainfixer.github.io/bin/](https://brainfixer.github.io/bin/)

## Правила безопасности
> [!CAUTION]
> Данный репозиторий является **публичным**. 
> Категорически нельзя загружать в него файлы, содержащие:
> * Приватные ключи (SSH, GPG и др.).
> * Токены авторизации и API-ключи.
> * Любую личную информацию или конфиденциальные конфигурации.

## Заметки
* В репозитории работает плагин [Github Pages Directory Listing](https://github.com/marketplace/actions/github-pages-directory-listing) для автоматической генерации и отображения списка файлов.
* Репозиторий предназначен для «неважных» файлов, не требующих защиты.
